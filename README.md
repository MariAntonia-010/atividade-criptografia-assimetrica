#  Criptografia Assimétrica com RSA

Este projeto demonstra o uso básico de criptografia assimétrica em Python utilizando a biblioteca "cryptography".

---

##  Conceito

A criptografia assimétrica utiliza um par de chaves:

-  Chave pública → usada para criptografar  
-  Chave privada → usada para descriptografar  

---

##  Funcionalidades

### 1. Confidencialidade
- A mensagem é criptografada com a chave pública  
- Apenas a chave privada consegue descriptografar  

### 2. Autenticidade
- A mensagem é assinada com a chave privada  
- A chave pública verifica a assinatura  

---

##  Tecnologias utilizadas

- Python  
- Biblioteca "cryptography"  

---

##  Execução

1. Instale a biblioteca:
- pip install cryptography
